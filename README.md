# Python CLI Typing Practice

ターミナル（コマンドプロンプト）上で動作する、シンプルなタイピング練習ツールです。
Pythonの標準ライブラリのみで動作するため、追加のインストール作業なしですぐに使えます。

特にプログラミングコードの写経（タイピング練習）に最適化されており、インデントの自動補完や入力文字のリアルタイム判定機能を備えています。

## 特徴

* **リアルタイム判定**: 入力したその瞬間に正誤判定を行い、色（緑/赤）でフィードバックします。
* **インデント自動処理**: 行頭の空白（インデント）は自動で入力されるため、文字の入力だけに集中できます。
* **クロスプラットフォーム**: Windows, Mac, Linux で動作します（Windowsの場合は `msvcrt`、Unix系は `termios` を自動判別）。
* **記録機能**: 練習終了後、日時・ミス回数・所要時間を `typing_log.csv` に自動保存します。
* **コンテキスト表示**: 現在入力している行の前後を表示し、全体の流れを見ながら練習できます。

## 必要要件

* Python 3.x
* OS: Windows / macOS / Linux

※ 外部ライブラリ（pip install）は不要です。

## セットアップと使い方

1.  **リポジトリのクローン**
    ```bash
    git clone [https://github.com/あなたのユーザー名/リポジトリ名.git](https://github.com/あなたのユーザー名/リポジトリ名.git)
    cd リポジトリ名
    ```

2.  **実行**
    初回実行時、自動的にサンプル問題が準備されます。
    ```bash
    python typing_practice.py
    ```

3.  **練習文の変更**
    フォルダ内に生成された `typing_practice.txt` を自由に書き換えてください。
    （このファイルはGitの管理対象外となっているため、自由に編集しても問題ありません）

4.  **操作方法**
    * 表示される文字通りにキーボードをタイピングしてください。
    * 間違った文字を入力すると背景が赤くなり、正しいキーを押すまで進めません。
    * 中断したい場合は `Ctrl + C` を押してください。

## ファイル構成

* `typing_practice.py`: スクリプト本体
* `typing_practice.txt`: 練習したい課題文（ユーザーが用意）
* `typing_log.csv`: 練習記録（実行後に自動生成されます）

## 記録ログについて

練習が完了すると、`typing_log.csv` に以下の形式で追記されます。

| Date | MissCount | Time(sec) |
| :--- | :--- | :--- |
| 2023/10/25 | 5 | 45.20 |
